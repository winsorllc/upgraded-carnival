# Chat Commands

Terminal chat-style command interface for agent interaction.

## Commands

- `/status` - Show system status
- `/help` - Show help
- `/skills` - List available skills
- `/health` - Quick health check

## Usage

```bash
/job/.pi/skills/chat-commands/chat.js /status
```
